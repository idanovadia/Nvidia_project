The purpose of this document is to describe the implementation of the Tiny URL software,
it designed to create short URL for personal and business usages.

Languages:
    The interface implemented in HTML, JS and CSS.
    The server implemented in Python using Flask lib for web framework.

Installing:
    TO know what to install to run this program u can see the requirements.txt file


Running:
    1.by using IDE like pycharm to run manually the server and the client
        the client rum on port 5000
    2. open index.html in browser
    3. insert your url
    4  get your new url
